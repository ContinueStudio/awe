## v0.3.7g — start.bat GBK 编码修复
- **Bug**: v0.3.7f 改完没测就发，老大跑起来一堆 `'PY' 不是内部或外部命令`
- **根因**: `set "PY=..."` 写在 `if exist` 块内被 cmd 拆 token；备注中文 GBK 错位
- **修复**:
  - `set "PY=..."` 提到外层
  - 备注行全部 ASCII
  - `set "ARGS=--window %*"` 预组参数
- **实测**: PowerShell 跑 `.\start.bat --skip-port-clean` → 200 OK
- **沉淀**: 修改 start.bat 后**必须实测**！这是教训
