### 关闭流程
1. 用户点击窗口 X 按钮 → Python `closing` 事件 → `evaluate_js('window.__showCloseConfirm()')` → 前端显示自定义对话框
2. 用户点击"退出" → `api.force_close()` → Python `window.destroy()`
3. 用户点击"取消" → 对话框关闭，保持窗口

### 技术要点
- Python `closing` 事件返回 `False` 阻止默认关闭
- 前端 `force_close` 确认后通过 pywebview API 通知 Python 销毁窗口
- `launch_window.py`（frameless）走 CustomTitleBar → `onClose` prop → 确认对话框 → force_close
